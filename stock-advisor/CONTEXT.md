# 股票顾问系统 (Stock Advisor)

A 股智能投资顾问系统。提供技术面、基本面、消息面的综合分析，自动生成评分排名、异动警报和买卖信号，支持策略回测与虚拟盘跟踪，通过 Web 面板和微信推送呈现。

## Language

### 领域概念

**股票池（Universe）**：
系统覆盖的股票范围。本项目为全量 A 股（5000+）。
_Avoid_: 股票列表、标的集

**筛选漏斗（Screening Funnel）**：
先粗筛再精析的两阶段流水线。日终全量粗筛选出候选，盘中只盯候选池异动。
_Avoid_: 过滤链、选股流程

**精选池（Selected Pool）**：
经日终粗筛后保留的股票，盘中对其进行深度实时监控。
_Avoid_: 候选池、观察列表、自选股

**信号（Signal）**：
系统的输出。分三类：评分排名信号、买卖点信号、异动警报。
_Avoid_: 推荐、提示、建议

**规则引擎（Rules Engine）**：
用户定义策略的方式。每条规则是一个条件组合，如 "MACD 金叉 AND 放量 AND PE < 30 → 买入信号"。所有逻辑透明可回溯。
_Avoid_: 策略系统、决策模块

**策略（Strategy）**：
一组规则的集合 + 权重配置。用户可在 Web 面板调整。
_Avoid_: 模型、算法

**回测（Backtest）**：
在历史数据上跑策略，验证收益曲线、最大回撤、夏普比率、胜率等指标。
_Avoid_: 模拟测试、历史验证

**虚拟盘（Paper Account）**：
按策略信号模拟下单，跟踪虚拟盈亏。用于验证策略有效性。
_Avoid_: 模拟账户、虚拟交易

**日终扫描（EOD Scan）**：
每天收盘后（~15:30）自动触发，更新数据 → 全量粗筛 → 深度分析 → 生成信号 → 推送。
_Avoid_: 收盘分析、每日跑批

**盘中异动（Intraday Alert）**：
交易时段对精选池实时监控，触发涨跌幅突破、成交量异常、关键位突破等条件时即时推送。
_Avoid_: 实时监控、盘中预警

### 信号类型

**评分排名信号（Ranking Signal）**：
综合技术面、基本面、消息面加权打分，输出 Top N。每日推送。
_Avoid_: 综合推荐、排名靠前

**买卖点信号（Entry/Exit Signal）**：
明确的交易建议，含方向（买入/卖出）、参考价格、止损位、止盈位。
_Avoid_: 交易建议、操作信号

**异动警报（Anomaly Alert）**：
事件驱动型通知：涨跌幅异动、成交量突增、突破关键位、财报暴雷等。
_Avoid_: 异常通知、突发事件

### 分析维度

**技术面（Technical）**：
基于 K 线数据的技术指标分析。经典 10 件套：MA, MACD, KDJ, RSI, BOLL, OBV, ATR, WR, CCI, BIAS。
_Avoid_: 技术指标、图表分析

**基本面（Fundamental）**：
基于财报数据的估值分析：PE, PB, ROE, 营收增速, 利润增速, 现金流等。
_Avoid_: 财务分析、价值分析

**消息面（Sentiment）**：
新闻、公告、研报、社交媒体舆情的情感分析和事件提取。
_Avoid_: 舆情、新闻分析

### 系统构成

**仪表盘（Dashboard）**：
Web 首页，展示大盘概览、精选池涨跌、今日信号一览、策略表现概览。
_Avoid_: 首页、总览

**个股深度页（Stock Detail）**：
单只股票的完整视图：K 线图 + 技术指标叠加 + 基本面数据 + 历史信号回顾。
_Avoid_: 个股分析、股票详情

**策略配置（Strategy Config）**：
Web 面板上调整策略参数：选择指标、设定权重、修改阈值。
_Avoid_: 参数设置、策略管理

**推送（Push）**：
信号分发渠道。实时信号走微信（Server酱 / PushPlus），浏览器通知作为补充。
_Avoid_: 通知、消息

## Relationships

- 日终扫描从**股票池（Universe）**中粗筛出**精选池（Selected Pool）**
- 盘中异动对**精选池**做实时监控，触发**异动警报（Anomaly Alert）**
- **规则引擎（Rules Engine）**执行**策略（Strategy）**，产生**信号（Signal）**
- **回测（Backtest）**用历史数据验证**策略**，结果反馈到**策略配置**调参
- **虚拟盘（Paper Account）**按**信号**模拟交易，跟踪虚拟盈亏
- 所有**信号**通过**推送（Push）**分发到微信和浏览器

## Example dialogue

> **用户**: "今天大盘跌了 3%，精选池里有没有抗跌的票？"
> **系统**: "精选池今日平均跌幅 1.2%，优于大盘。其中 XX 医药 逆势涨 2.3%，触发异动警报：放量突破 60 日均线。评分排名仍为 Top 5。"
>
> **用户**: "那我看看它的基本面。"
> **系统**: "XX 医药 PE 22.3（行业均值 35），ROE 18%，连续 3 季营收增速 >20%。最近一条公告：新药获批临床试验。"
>
> **用户**: "可以买吗？"
> **系统**: "当前策略未触达买入信号。MACD 刚金叉但 RSI 已到 72（超买区），建议等回调。可在策略配置中调低 RSI 阈值。"

## Flagged ambiguities

- "自选股" vs "精选池" — 已解决：自选股是用户手动添加的，精选池是系统日终扫描自动选出的，两者不同。
- "策略" vs "规则" — 已解决：规则（Rule）是单个条件组合，策略（Strategy）是规则集合 + 权重配置。
