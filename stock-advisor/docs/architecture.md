## 技术选型

| 层 | 技术 | 理由 |
|---|---|---|
| 后端框架 | FastAPI + uvicorn | 异步、高性能、自动 OpenAPI |
| 前端 | Vue 3 + Element Plus | 国内生态好，组件丰富 |
| 数据源 | akshare | 免费、覆盖全、无需 API Key |
| 技术指标 | ta-lib / pandas-ta | 经典 10 件套 |
| 数据库 | SQLite + SQLAlchemy | 零配置启动，够用 |
| 调度 | APScheduler | Python 原生，支持 cron |
| 微信推送 | Server酱 / PushPlus | 免费额度够个人用 |
| K 线图 | ECharts | 交互式、支持叠加指标 |

## 数据流

```
收盘后 (15:30)
  EOD调度触发
  → 数据采集: akshare拉取当日K线 + 财报 + 新闻
  → 全量粗筛: 5000+ → 成交量>阈值 + 剔除ST + 剔除停牌
  → 深度分析: 候选池 → 技术面 + 基本面 + 消息面打分
  → 信号生成: 规则引擎评估 → 信号入库
  → 推送: Top 10 排名 + 异动警报 → 微信

交易时段 (9:30-15:00)
  盘中监控 (每5分钟)
  → 精选池实时行情
  → 异动检测: 涨跌幅/量比/突破判断
  → 触发时即时推送
```
